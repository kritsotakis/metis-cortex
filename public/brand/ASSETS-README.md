## Lockup rules

- **Clear space** equals the height of the "CORTEX" subline (stacked) or the wordmark cap-height (horizontal).
- **Minimum size:** mark ≥ 32px, horizontal lockup ≥ 120px wide, stacked lockup ≥ 96px wide.
- **Bronze rule.** The single bronze brushstroke inside the mark is the only place bronze appears on the lockup. Don't recolour the wordmark bronze.
- **Don't rotate, recolour, drop-shadow, gradient-fill or stroke the mark.** If you need it on a busy photo, use `metiscortex-mark-transparent.png` over a 60% Ink Navy scrim.
- **Cream wordmark on dark, Ink Navy wordmark on light.** Never bronze.
- **No "MC" monogram.** The full wordmark is non-negotiable above 96px.
